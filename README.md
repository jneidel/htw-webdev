## Stack

- Node (backend)
- Express (http server)
- MariaDB (SQL database)
- Vue@3 (spa framework)
- jest (test runner)
- pug (html template)
- eslint (linter)

## Installation

```sh
npm install
```

Install database

```sh
yay -S mariadb
```

## Usage

Start the server:

```sh
npm start
npm run start-prod
```

Start the database:

```sh
sudo systemctl start mysql
```

## Run tests

```sh
npm test
```
