## Stack

- Node (backend)
- Express (http server)
- Vue@3 (spa framework)
- SQL (database)
- jest (test runner)
- pug (html template)
- eslint (linter)

## Installation

```sh
npm install
```

## Usage

Start the server:

```sh
npm start
```
